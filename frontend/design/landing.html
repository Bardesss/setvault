// what is setvault

The vault for sets your crew actually listens to.

Not Spotify, not Mixcloud, not 1001tracklists — but every part of those that matters when you're trying to find a specific 90-second moment in a 2-hour set, or you want to know what that ID was at 1:02:34.

1

Yours, on your hardware

One docker compose up on a NUC or homeserver. Files stay on disk you own. The app keeps working when the WAN drops.

2

Built for sets, not tracks

Time-coded tracklists are first-class. "ID — ID" is supported. Mashups link to both source tracks. Drop-bookmarks survive forever.

3

Your existing players, native

DLNA on Sonos and Kodi. Subsonic API for Substreamer / Symfonium. RSS feeds with chapter-marker tracklists for any podcast app.

Player & tracklist

The marquee — a player that actually respects what a live set is.

Waveform with playhead and clickable tracklist sidebar. Time-stamped comments and bookmarks pinned to the wave. Variable-speed playback with pitch preservation. Keyboard-native: M to mark a track at the playhead, [/] to step entries, Cmd-K for global search.

Tracklist editor

Three ways to enter a tracklist.

Live: hit M during playback. Paste: drop forum text, an LLM parses it. URL: 1001tracklists scraper builds the whole thing. Every entry tracks "raw / resolved / AcoustID-confirmed" so you always know what's verified.

// quickstart

From git clone to "the crew is listening" in 60 seconds.

Three commands. No registration required. Default config works on a fresh box; OIDC, providers, and SMTP are admin-configured later in the UI.

  1. Clone the repo and copy the example compose file.
  2. Set SECRET_KEY in .env.
  3. docker compose up -d. Open the URL printed at the end.
  4. Create the admin account on the first-run wizard. Invite the rest of your crew.
setvault@nuc:~ 
# clone the repo
$ git clone https://github.com/yourname/setvault.git
$ cd setvault/infra/docker

# bootstrap config
$ cp compose.example.yml compose.yml
$ echo "SECRET_KEY=$(openssl rand -hex 32)" > .env

# launch
$ docker compose up -d
