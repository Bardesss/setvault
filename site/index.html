SetVault

Self-hosted vault for DJ live sets.

Lossless live-set storage. Time-coded tracklists. Provider-enriched metadata. Comments with waveform markers. Per-user RSS feeds. An embeddable player. An installable PWA. All running on your server, behind your reverse proxy, under your GPL-3.0 license.

Mixcloud × 1001tracklists × Plex — but for DJ live sets specifically, not for individual tracks.

Self-host in three commands

curl -L -o compose.yml https://raw.githubusercontent.com/Bardesss/setvault/main/infra/docker/compose.example.yml
curl -L -o .env       https://raw.githubusercontent.com/Bardesss/setvault/main/.env.example
# edit .env: SECRET_KEY, POSTGRES_PASSWORD, BASE_URL
docker compose pull && docker compose up -d

Multi-arch (linux/amd64 + linux/arm64) images live at ghcr.io/bardesss/setvault-web and ghcr.io/bardesss/setvault-worker. Every release is signed with cosign and ships a CycloneDX SBOM.

Honest about AI

SetVault is built with substantial help from Claude Code. Every change is reviewed, tested, and shipped by the human maintainer — but the bulk of the code is co-authored with an LLM, and the commits are marked accordingly.
